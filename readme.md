# Раскладка продуктов в поход

Калькулятор для планирования раскладки продуктов в туристический поход. Позволяет рассчитать необходимое количество продуктов на группу с учётом количества человек, дней и приёмов пищи.

## Возможности

- Раскладка по дням с независимыми наборами приёмов пищи для каждого дня
- Четыре приёма пищи: завтрак, обед, перекус, ужин (можно включать/отключать для каждого дня отдельно)
- Поля "Время" и "Км" для каждого приёма пищи (опционально)
- Автоматический подсчёт итогов по всем продуктам
- Шпаргалка с типичными походными нормами продуктов (на 1 человека на 1 приём)
- Автоматический пересчёт при изменении количества людей
- Таблица для печати с разбивкой по дням
- Экспорт/импорт раскладки в JSON
- Автосохранение в localStorage браузера
- Адаптивный дизайн для мобильных устройств

## Технологии

Проект реализован на чистом HTML, CSS и JavaScript без использования фреймворков и сборщиков. Это single-page application, которое можно развернуть на любом статическом хостинге.

- HTML5
- CSS3 (Grid, Flexbox, CSS Variables)
- Vanilla JavaScript (ES6+)
- localStorage для сохранения данных

## Структура проекта
```bash
food.naviu.ru/
├── index.html # Основной файл приложения
└── typical.js # Справочник типичных походных продуктов и норм
```

### Локальный запуск

Просто откройте `index.html` в браузере. Никакой сборки не требуется.

## Единицы измерения
Поддерживаются следующие единицы:
```bash
g — граммы
kg — килограммы
pcs — штуки
pack — упаковки
can — банки
```

При итоговом подсчёте граммы и килограммы конвертируются автоматически.

## Шпаргалка норм
Файл typical.js содержит справочник типичных походных продуктов с нормами на 1 человека на 1 приём пищи. Нормы усреднены по классическим туристическим раскладкам.
Чтобы изменить нормы, отредактируйте массив TYPICAL в файле `typical.js`.
